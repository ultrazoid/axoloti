Compiling from source on Linux

Assuming Ubunut Linux. Other Distributions may need minor changes.

Download

From GIT

Open a terminal and enter:

$> git clone https://github.com/axoloti/axoloti.git

Install

Root privileges are required to add a udev rule to grant access to Axoloti Core on USB. open a terminal and enter:

$> cd axoloti
$> cd platform_linux
$> ./build.sh
